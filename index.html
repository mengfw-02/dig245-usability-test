Translation: Task 1

Description

User is given the task of changing an article from English to Spanish. There is several ways of doing this, but the intended way from Wikipedia is to use the language option on the side, see video below.

Experience
  1. User was not able to find the option from the start.
  2. User ultimately decided to scroll down to change the language instead of using the side bar option.
Time Taken

42.83 seconds üëé

Recommendation
  • Consider adding a language option at the top of the page, because users who don't understand the current language will feel very overwhelmed to even continue reading, thus they will exit the website.

Research: Task 3

Description
